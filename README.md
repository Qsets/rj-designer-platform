# 钧鉴设计师平台 (RJ Designer Platform)\n\n一个专业的设计师与业主交易平台，提供设计作品展示、交易、沟通等全方位服务。\n\n## 🚀 项目特色\n\n- **现代化技术栈**: React 18 + TypeScript + Spring Boot 3\n- **响应式设计**: 支持桌面端和移动端\n- **安全可靠**: JWT认证、数据加密、安全防护\n- **高性能**: Redis缓存、数据库优化、CDN加速\n- **容器化部署**: Docker + Docker Compose\n- **监控完善**: Prometheus + Grafana\n\n## 📋 功能模块\n\n### 用户系统\n- ✅ 用户注册/登录\n- ✅ 个人资料管理\n- ✅ 角色权限控制（设计师/业主/管理员）\n- 🔄 实名认证\n- 🔄 信用评级系统\n\n### 设计师功能\n- 🔄 作品集管理\n- 🔄 服务项目发布\n- 🔄 订单管理\n- 🔄 收益统计\n- 🔄 客户沟通\n\n### 业主功能\n- 🔄 需求发布\n- 🔄 设计师搜索\n- 🔄 作品浏览\n- 🔄 订单跟踪\n- 🔄 评价系统\n\n### 交易系统\n- 🔄 在线支付（支付宝/微信）\n- 🔄 资金托管\n- 🔄 交易保障\n- 🔄 发票管理\n\n### 管理后台\n- 🔄 用户管理\n- 🔄 订单管理\n- 🔄 财务管理\n- 🔄 数据统计\n- 🔄 系统配置\n\n## 🛠 技术架构\n\n### 前端技术栈\n- **框架**: React 18 + TypeScript\n- **UI库**: Ant Design 5.x\n- **状态管理**: Zustand\n- **路由**: React Router 6\n- **HTTP客户端**: Axios\n- **构建工具**: Vite\n- **样式**: CSS Modules + Less\n\n### 后端技术栈\n- **框架**: Spring Boot 3.x\n- **数据库**: MySQL 8.0\n- **缓存**: Redis 7.x\n- **安全**: Spring Security + JWT\n- **文档**: OpenAPI 3 (Swagger)\n- **构建工具**: Maven\n\n### 基础设施\n- **容器化**: Docker + Docker Compose\n- **反向代理**: Nginx\n- **监控**: Prometheus + Grafana\n- **日志**: ELK Stack (可选)\n- **CI/CD**: GitHub Actions (计划中)\n\n## 📦 项目结构\n\n```\nrj-designer-platform/\n├── frontend/                 # 前端React应用\n│   ├── src/\n│   │   ├── components/      # 公共组件\n│   │   ├── pages/          # 页面组件\n│   │   ├── stores/         # 状态管理\n│   │   ├── utils/          # 工具函数\n│   │   └── types/          # TypeScript类型定义\n│   ├── public/             # 静态资源\n│   └── package.json\n├── backend/                 # 后端Spring Boot应用\n│   ├── src/main/java/      # Java源码\n│   ├── src/main/resources/ # 配置文件\n│   └── pom.xml\n├── nginx/                   # Nginx配置\n├── scripts/                 # 部署脚本\n├── docker-compose.yml       # 开发环境\n├── docker-compose.prod.yml  # 生产环境\n└── README.md\n```\n\n## 🚀 快速开始\n\n### 环境要求\n\n- Node.js 18+\n- Java 17+\n- MySQL 8.0+\n- Redis 7.0+\n- Docker & Docker Compose (可选)\n\n### 本地开发\n\n1. **克隆项目**\n   ```bash\n   git clone https://github.com/Qsets/rj-designer-platform.git\n   cd rj-designer-platform\n   ```\n\n2. **安装依赖**\n   ```bash\n   npm run install:all\n   ```\n\n3. **配置数据库**\n   - 创建MySQL数据库 `junjian_platform`\n   - 导入初始化SQL脚本\n   - 配置 `backend/src/main/resources/application.yml`\n\n4. **启动开发服务**\n   ```bash\n   npm run dev\n   ```\n\n   访问地址：\n   - 前端: http://localhost:3000\n   - 后端API: http://localhost:8080\n   - API文档: http://localhost:8080/swagger-ui.html\n\n### Docker部署\n\n1. **开发环境**\n   ```bash\n   docker-compose up -d\n   ```\n\n2. **生产环境**\n   ```bash\n   # 配置环境变量\n   cp .env.example .env\n   \n   # 生成密钥文件\n   ./scripts/deploy.sh\n   \n   # 启动服务\n   docker-compose -f docker-compose.prod.yml up -d\n   ```\n\n## 🔧 配置说明\n\n### 环境变量\n\n创建 `.env` 文件并配置以下变量：\n\n```env\n# 数据库配置\nMYSQL_ROOT_PASSWORD=your_root_password\nMYSQL_PASSWORD=your_password\nREDIS_PASSWORD=your_redis_password\n\n# JWT配置\nJWT_SECRET=your_jwt_secret\n\n# 支付配置\nALIPAY_APP_ID=your_alipay_app_id\nWECHAT_APP_ID=your_wechat_app_id\n\n# 邮件配置\nMAIL_HOST=smtp.example.com\nMAIL_USERNAME=your_email\nMAIL_PASSWORD=your_email_password\n```\n\n### SSL证书\n\n生产环境建议使用正式SSL证书：\n\n```bash\n# 使用Let's Encrypt\ncertbot --nginx -d yourdomain.com\n\n# 或者放置证书文件到\nnginx/ssl/server.crt\nnginx/ssl/server.key\n```\n\n## 📊 监控和日志\n\n### Prometheus监控\n\n访问 http://localhost:9090 查看Prometheus监控面板\n\n### Grafana仪表板\n\n访问 http://localhost:3001 查看Grafana仪表板\n- 默认用户名: admin\n- 密码: 查看 `secrets/grafana_password.txt`\n\n### 日志查看\n\n```bash\n# 查看所有服务日志\ndocker-compose logs -f\n\n# 查看特定服务日志\ndocker-compose logs -f backend\ndocker-compose logs -f frontend\n```\n\n## 🔒 安全配置\n\n运行安全配置脚本：\n\n```bash\nsudo ./scripts/security-setup.sh\n```\n\n定期运行安全检查：\n\n```bash\nsudo ./scripts/security-check.sh\n```\n\n## 🧪 测试\n\n```bash\n# 前端测试\ncd frontend\nnpm run test\n\n# 后端测试\ncd backend\n./mvnw test\n```\n\n## 📈 性能优化\n\n### 前端优化\n- 代码分割和懒加载\n- 图片压缩和WebP格式\n- CDN加速\n- 浏览器缓存策略\n\n### 后端优化\n- 数据库索引优化\n- Redis缓存策略\n- 连接池配置\n- JVM参数调优\n\n### 数据库优化\n- 查询优化\n- 索引策略\n- 分库分表（规划中）\n- 读写分离（规划中）\n\n## 🤝 贡献指南\n\n1.
